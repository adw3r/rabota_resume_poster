Основные обязанности: 
Разработка и поддержка скриптов для размещения рекламных объявлений, фарм аккаунтов, парсинг сайтов.

Используемое ПО:
Python3, SQL, Postgresql, Selenium webdriver, Git, Docker, Pycharm, Multilogin, Postman, MS Excel, Linux Ubuntu server, Windows 10,
Rucaptcha API, OnlineSim API, SmsActivate API, proxy

Используемые библиотеки:
pandas, requests, selenium, telethon, psycopg2, instapy, instagrapi.
